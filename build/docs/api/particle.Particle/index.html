particle.Particle

包    particle

类    public class Particle

子类    particle.GravityParticle

#公共属性

属性
alpha : number
表示粒子的 Alpha 透明度值
blendMode : number
表示粒子的混合模式
currentTime : number
表示粒子当前存活时间，以毫秒为单位，取值范围(0,Number.MAX_VALUE]，该值超过 totalTime 时，粒子将会被销毁
rotation : number
表示 Particle 实例距其原始方向的旋转程度，以度为单位
scale : number
表示从注册点开始应用的对象的缩放比例（百分比）
totalTime : number
表示粒子的存活总时间，以毫秒为单位，取值范围(0,Number.MAX_VALUE]
x : number
表示 Particle 实例相对于父级本地坐标的 x 坐标
y : number
表示粒子实例相对于父级本地坐标的 y 坐标
