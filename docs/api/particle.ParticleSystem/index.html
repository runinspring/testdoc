particle.ParticleSystem

包    particle

类    public class ParticleSystem

子类    particle.GravityParticleSystem

#公共属性

属性
emissionRate : number
表示粒子出现间隔，单位毫秒，取值范围(0,Number.MAX_VALUE]
emissionTime : number
表示粒子出现总时间，单位毫秒，取值范围(0,Number.MAX_VALUE]，-1表示无限时间
emitterBounds : egret.Rectangle
表示当前粒子系统中发射粒子的渲染边界范围，使用以发射点为基准的坐标系
emitterX : number
表示粒子出现点X坐标，取值范围[-Number.MAX_VALUE,Number.MAX_VALUE]
emitterY : number
表示粒子出现点Y坐标，取值范围[-Number.MAX_VALUE,Number.MAX_VALUE]
maxParticles : number
表示粒子系统最大粒子数，超过该数量将不会继续创建粒子，取值范围[1,Number.MAX_VALUE]
particleClass : any
表示粒子类，如果设置创建粒子时将创建该类
texture : egret.Texture
表示粒子所使用的纹理

#emitterBounds

emitterBounds : egret.Rectangle

  • 支持版本:all
  • 运行平台:Web,Native

#emitterX

emitterX : number = 0

  • 支持版本:all

#emitterY

emitterY : number = 0

  • 支持版本:all
  • 运行平台:Web,Native

表示粒子出现点Y坐标，取值范围[-Number.MAX_VALUE,Number.MAX_VALUE]

#maxParticles

maxParticles : number = 200

  • 支持版本:all
  • 运行平台:Web,Native

表示粒子系统最大粒子数，超过该数量将不会继续创建粒子，取值范围[1,Number.MAX_VALUE]
